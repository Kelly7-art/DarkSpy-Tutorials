TikTok Hacking

Lesson Overview

This tutorial shows how to hackers use a simple phishing tool to hack Snapchat accounts.

📱 Applications Used

Download the required apps used in this tutorial.

⬇ ClipTok App ⬇ MT Manager

Use these Tools for Entertainment and Educational Purposes only!

Steps Taken

  • Install ClipTok, MT Manager and Telegram on your device.
  • Open MT Manager and Tap on the menu button, then tap on Extract APK.
  • Search For, and Select ClipTok > Extract APK
  • After Extracting, tap on locate to open where the extracted APK is located.
  • Tap on the Extracted APK and tap on view.
  • Tap on the "assets" folder and open the "Input Bot Token.txt"
  • Open Telegram and create a new bot from @botfather and copy the bot token
  • Open your bot and tap start.
  • Open MT Manager and paste your bot token in the "Input Bot Token.txt" file and save
  • Open "Input Telegram ID.txt" file from MT Manager.
  • Open Telegram and search for "IDBOT"
  • Start "IDBOT" and copy your User ID (UID)
  • Open MT Manager and paste your bot token in the "Input Bot Token.txt" file and save
  • Go back and reinstall the app from MT Manager
  • Open ClipTok to Test it. Credentials must be sent to your Telegram bot
  • Hackers usually share the modded app to target especially in groupchats
